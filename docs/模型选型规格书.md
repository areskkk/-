# 模型选型规格书

> 依据：`memory/model-selection-standards.md`、`PRD改进.md`、prd-planner 访谈结果。  
> 口径：混合部署，OpenAI-compatible Chat / Embedding / Speech 抽象。候选模型用于 POC，不代表已采购或已部署。

## 1. 选型原则

| 原则 | 要求 |
| --- | --- |
| 数据安全 | 企业申报材料、财务、用工、审批数据走私有化或专有云模型 |
| 分层使用 | 低敏高频任务用低成本模型，高风险任务用高可靠模型 |
| 规则优先 | 资格预判和审核辅助必须结合规则引擎，不单独依赖模型 |
| 可解释 | 政策问答、预判、审核草稿必须带政策或材料引用 |
| 可替换 | 模型服务统一按 OpenAI-compatible 接口抽象 |
| 可审计 | 每次模型调用记录 trace_id、输入摘要、输出摘要、模型名、版本、耗时 |

## 2. 部署模式

| 任务类型 | 部署位置 | 原因 |
| --- | --- | --- |
| 公开政策问答、Query 改写、意图识别 | 云 API 或低成本兼容模型 | 数据低敏、调用高频、重视成本和延迟 |
| 企业画像、申报材料、财税/用工数据处理 | 私有化或专有云模型 | 涉企业敏感数据 |
| 资格预判、审批辅助、审批意见草稿 | 私有化或专有云高可靠模型 | 涉审核建议和合规风险 |
| Embedding/Reranker | 优先私有化或专有云 | 政策库和企业上下文需权限过滤 |
| Speech-to-Text | 可混合部署 | 语音内容若包含敏感申报信息，应走私有化或专有云 |

## 3. 候选模型矩阵

| 能力 | 候选模型/服务 | 部署建议 | POC 关注 |
| --- | --- | --- | --- |
| 低延迟 Chat | Qwen 系列、DeepSeek-V3、OpenAI-compatible Chat | 云 API 或专有云 | P95、结构化输出成功率 |
| 强推理 Chat | DeepSeek-R1、Qwen 高参数推理模型、OpenAI-compatible reasoning model | 私有化/专有云优先 | 复杂条件拆解、延迟、成本 |
| 审批辅助 Chat | DeepSeek-R1、Qwen 高可靠模型、OpenAI-compatible high reliability model | 私有化/专有云 | 引用完整率、人工采纳率 |
| Embedding | BGE Embedding、OpenAI-compatible embedding | 私有化/专有云优先 | Top3 命中率、维度、索引成本 |
| Reranker | BGE Reranker、其他兼容 rerank 服务 | 私有化/专有云优先 | 重排准确率、延迟 |
| Speech-to-Text | Whisper、PaddleSpeech、OpenAI-compatible speech | 混合部署 | 中文口音、噪声、敏感数据边界 |
| OCR 后字段抽取 | Qwen/DeepSeek 结构化输出模型、专用抽取模型 | 私有化/专有云 | 字段准确率、JSON 合法率 |

## 4. 按 Agent 分配

| Agent | 主要任务 | 模型能力 | 部署建议 |
| --- | --- | --- | --- |
| 企业诉求 Agent | 意图识别、槽位填充、追问 | 低延迟 Chat，稳定 JSON 输出 | 低敏场景可云 API，含企业敏感上下文时走专有云 |
| 检索调度 Agent | Query 改写、检索策略 | Embedding + Reranker + 低成本 Chat | Embedding/Reranker 优先私有化或专有云 |
| 政策分析 Agent | 条件解释、差距说明 | 规则引擎优先，强推理 Chat 辅助解释 | 私有化/专有云 |
| 申报辅助 Agent | 表单预填、材料映射 | 结构化输出 Chat | 私有化/专有云 |
| 审核 Agent | 预审建议、风险点、意见草稿 | 高可靠 Chat + 规则校验 | 私有化/专有云，人工确认 |

## 5. POC 验收指标

| 指标 | 目标 |
| --- | --- |
| 政策问答答案引用率 | >= 95% |
| 检索 Top3 命中率 | >= 85% |
| 资格预判准确率 | 初始目标 >= 85%，POC 后校准 |
| 结构化输出合法率 | >= 98% |
| 审批草稿人工采纳率 | 二期目标 >= 60% |
| 政策问答 P95 | <= 5s |
| 资格预判 P95 | <= 8s，复杂任务可异步 |
| OCR 后字段抽取置信度 | 低于 0.85 转人工 |

## 6. 禁止项

- 不得把企业申报材料、财务、用工、审批数据默认发送到不受控公网模型。
- 不得把候选模型写成已部署事实。
- 不得让模型输出替代最终审批决定。
- 不得在无引用时输出正式政策答案。
