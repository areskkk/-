# 移动端适配方案

> 已确认：MVP 手机端采用 H5 响应式，支持完整申报。

## 1. 适配目标

- 手机端完成政策查询、资格预判、表单填写、材料上传、提交、补正。
- 与桌面端共用业务 API 和表单 schema。
- 不单独建设小程序。

## 2. 断点

| 断点 | 范围 | 布局 |
| --- | --- | --- |
| mobile | < 768px | 单列、底部固定操作栏 |
| tablet | 768-1023px | 单列或双列 |
| desktop | >= 1024px | 多列、表格、分屏 |

## 3. 关键交互

| 场景 | 手机端要求 |
| --- | --- |
| 政策列表 | 卡片化，突出符合度和截止时间 |
| 申报表单 | 分步填写，自动保存草稿 |
| 材料上传 | 支持拍照、相册、文件选择 |
| OCR 结果 | 字段级确认，低置信度高亮 |
| 补正 | 展示原因、截止时间、上传入口 |

## 4. 验收标准

- 手机端可完成完整申报闭环。
- 关键按钮不被键盘遮挡。
- 上传失败保留草稿。
- 长表单支持分步保存。
- 关键页面通过截图和布局检查。
