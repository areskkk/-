# 部署架构文档

> 口径：普通云部署，云厂商中立，不绑定具体云厂商产品名。

## 1. 逻辑架构

```text
CDN/负载均衡
  ├─ Web 前端服务
  ├─ API 网关
  │   ├─ 业务服务
  │   ├─ RAG/Agent 服务
  │   ├─ OCR 任务服务
  │   └─ 后台管理服务
  ├─ 关系型数据库
  ├─ 对象存储
  ├─ 向量数据库
  ├─ 缓存/队列
  └─ 日志与监控
  └─ 模型网关
      ├─ 云API兼容模型出口（低敏任务）
      └─ 私有化/专有云模型服务（敏感任务）
```

## 2. 环境

| 环境 | 用途 |
| --- | --- |
| dev | 开发联调 |
| test | 测试验收 |
| staging | 上线前演练 |
| prod | 正式环境 |

## 3. 资源建议

| 组件 | 要求 |
| --- | --- |
| Web/API | 可水平扩容 |
| 数据库 | 主备或托管高可用 |
| 对象存储 | 存放政策文件、申报材料、OCR 原图 |
| 向量库 | 政策检索 |
| 队列 | OCR、异步预判、通知 |
| 日志 | 应用日志、审计日志、模型 trace |
| 模型网关 | 统一 OpenAI-compatible Chat / Embedding / Speech 接口 |
| Postgres Checkpoint | 保存 LangGraph 状态和恢复点 |

## 4. 网络与安全

- 前端只访问 API 网关。
- 数据库、对象存储、向量库不直接暴露公网。
- 管理后台限制访问来源和角色权限。
- 敏感文件下载需鉴权和短期签名 URL。
- 企业申报材料、财务、用工、审批数据不得默认发送到不受控公网模型。
- 云 API 出口仅允许低敏任务，并记录模型调用审计。

## 5. Agent 与模型部署

| 组件 | 部署口径 |
| --- | --- |
| Agent 服务 | 业务内网服务，调用模型网关和工具服务 |
| 模型网关 | 对上提供 OpenAI-compatible 接口，对下路由云 API、专有云或私有化模型 |
| LangGraph Checkpoint | Postgres 存储，支持长流程恢复 |
| Embedding/Reranker | 优先专有云或私有化，避免政策库和权限上下文外泄 |
| Speech-to-Text | 低敏语音可走云 API，敏感语音走专有云或私有化 |

## 6. 备份

| 数据 | 频率 | 保留 |
| --- | --- | --- |
| 数据库 | 每日全量 + 持续增量 | 不少于 30 天 |
| 对象存储 | 每日备份 | 不少于 30 天 |
| 审计日志 | 每日归档 | 按制度确认 |
