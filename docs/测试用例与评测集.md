# 测试用例与评测集

## 1. 资格预判用例

| 类型 | 用例 |
| --- | --- |
| 正例 | 企业画像完整，满足所有硬性条件 |
| 反例 | 行业、区域、规模等硬性条件不满足 |
| 缺字段 | 缺纳税额、员工数、出口额等关键字段 |
| 材料异常 | 材料过期、类型不匹配、OCR 低置信度 |
| 政策冲突 | 条款互斥或存在自由裁量 |

要求：MVP 至少覆盖 20 条高频政策，每条包含正例、反例、缺字段例。

### 1.1 开发自测样例

> 说明：以下为脱敏开发样例，用于验证规则引擎、字段映射、OCR Schema 和 Agent 输出，不代表真实政策结论。真实上线评测集需由政策管理员基于首批政策核验后补齐。

| 用例 ID | 场景 | 输入摘要 | 规则条件 | 预期结果 |
| --- | --- | --- | --- | --- |
| ELG-001 | 家具行业正例 | region=南康区；industry=家具制造；revenue_amount=8000000；business_license 已上传且 OCR 企业名称一致 | 行业属于家具制造；营收 >= 500 万；营业执照存在；OCR 名称一致 | eligible |
| ELG-002 | 区域硬性不符合 | region=章贡区；industry=家具制造；revenue_amount=9000000 | region 必须等于南康区 | ineligible |
| ELG-003 | 缺少纳税额 | region=南康区；industry=家具制造；tax_amount 缺失 | tax_amount >= 100000 且 required=true | need_info |
| ELG-004 | OCR 低置信度 | 营业执照 OCR credit_code confidence=0.72；企业手填 credit_code 格式正确 | OCR 关键字段置信度需 >= 0.85 | manual_review |
| ELG-005 | 技改或出口 OR 条件 | tech_upgrade_status=none；export_amount=2000000；contract 已上传 | tech_upgrade_status in [in_progress, completed] OR export_amount >= 1 | eligible |

## 2. 政策问答评测集

| 类型 | 要求 |
| --- | --- |
| 事实问答 | 必须引用政策来源 |
| 资格咨询 | 必须追问缺失字段 |
| 无结果 | 不得编造，进入人工兜底 |
| 过期政策 | 标记废止或过期 |

## 3. 权限测试

- 企业只能查看本企业数据。
- 政府人员只能查看授权事项和材料字段。
- 后台用户只能按角色操作。
- Agent 审核记录不可被普通用户修改。

## 4. 手机端测试

- H5 可完整申报。
- 拍照上传成功。
- 上传失败保留草稿。
- 补正可重新提交。

## 5. 安全测试

- 日志脱敏。
- 文件下载鉴权。
- 越权访问拦截。
- 审计日志不可篡改。
