# 低置信度阈值定义

> 口径：初始阈值用于 MVP 和 POC，必须基于真实政策和材料样本校准。

## 1. 阈值表

| 类型 | 指标 | 初始阈值 | 处理 |
| --- | --- | --- | --- |
| 检索 | Top1 相似度 | < 0.75 | 不给确定答案，扩大检索或转人工 |
| OCR | 字段置信度 | < 0.85 | 人工确认 |
| 规则 | 必填证据完整度 | 缺失任一必填证据 | 输出待补充 |
| 模型 | 引用完整性 | 无政策或材料引用 | 转人工 |
| 模型 | 与规则一致性 | 与规则冲突 | 转人工 |

## 2. 人工兜底

| 项 | 规则 |
| --- | --- |
| 队列归属 | 平台运营统一队列 |
| 普通任务 SLA | 1 个工作日内处理 |
| 审核争议 | 平台运营转责任部门 |
| 处理结果 | 回流知识库和评测集 |

## 3. 校准机制

- POC 阶段抽取不少于 20 条高频政策和真实/脱敏材料样本。
- 每周统计误判、漏判、人工采纳率。
- 阈值调整必须记录版本、调整人、原因和生效时间。
