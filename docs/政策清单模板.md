# 政策清单模板

> 用途：首批约 60 份政策的导入和结构化准备。  
> 来源规则：官方来源可作为正式候选；权威媒体来源必须标记“待官方原文核验”。

## 1. 清单字段

| 字段 | 说明 | 示例 |
| --- | --- | --- |
| policy_id | 内部编号 | NK-POLICY-001 |
| policy_title | 政策标题 | 关于支持家具产业高质量发展的若干措施 |
| source_type | 来源类型 | official、authoritative_media |
| source_name | 来源名称 | 南康区人民政府 |
| source_url | 来源链接 | https://... |
| publish_date | 发布日期 | 2025-01-01 |
| responsible_department | 责任部门 | 工信局 |
| applicable_objects | 适用对象 | 家具制造企业 |
| policy_category | 政策类别 | 奖补、技改、融资、用工、出口 |
| application_window | 申报时间 | 常年/具体时间 |
| key_conditions | 关键条件 | 行业、营收、纳税、员工、技改 |
| required_materials | 材料要求 | 营业执照、财报、合同 |
| status | 状态 | candidate、verified、effective、revoked |
| verification_status | 核验状态 | 已核验、待官方原文核验 |
| notes | 备注 | 需业务确认 |

## 2. 导入要求

- `source_url` 必填。
- 权威媒体来源不得直接作为最终政策原文。
- 政策入库前必须由政策管理员核验责任部门、有效期、申报条件和材料。
- 政策结构化后必须进入审核发布流程。
