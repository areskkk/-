# 前端页面路由与组件树

> 前端形态：桌面 Web + H5 响应式手机端。  
> 设计验收：关键页面参考 Pencil MCP 或等价工具截图/布局检查。

## 1. 企业端路由

| 路由 | 页面 | 关键组件 |
| --- | --- | --- |
| /enterprise/dashboard | 企业工作台 | PolicyRecommendationList、TodoList、ProgressSummary、SupplementAlert |
| /enterprise/qa | 政策问答 | ChatInput、CitationAnswer、FollowUpQuestion、EligibilityCard |
| /enterprise/profile | 企业画像 | ProfileForm、OCRLicenseCheck、FieldCompletion |
| /enterprise/policies | 政策推荐 | PolicyFilter、PolicyCard、EligibilityBadge |
| /enterprise/applications/new | 新建申报 | PolicySelector、MergedForm、MaterialUploader、OCRResult |
| /enterprise/applications/:id | 申报详情 | ApplicationTimeline、PolicyItemStatus、MaterialList |
| /enterprise/supplements/:id | 补正 | SupplementReason、MaterialFixForm、SubmitBar |

## 2. 政府端路由

| 路由 | 页面 | 关键组件 |
| --- | --- | --- |
| /gov/tasks | 审核任务 | TaskTable、StatusFilter、DeadlineWarning |
| /gov/tasks/:itemId | 审核详情 | PolicyConditionCompare、MaterialPreview、AISuggestion、DecisionPanel |
| /gov/manual-review | 人工复核 | ManualReviewQueue、ReasonTag、ResolvePanel |
| /gov/statistics | 部门统计 | ReviewDurationChart、SupplementRateChart |

## 3. 管理后台路由

| 路由 | 页面 | 关键组件 |
| --- | --- | --- |
| /admin/policies | 政策库 | PolicyTable、ImportButton、VersionTag |
| /admin/policies/:id/schema | 政策结构化 | PolicyOriginalViewer、ConditionEditor、MaterialRequirementEditor |
| /admin/permissions | 权限管理 | DepartmentTree、RoleMatrix、MaterialPermission |
| /admin/fallback | 人工兜底 | FallbackQueue、AnswerEditor、QualityCheck |
| /admin/model-config | 模型与阈值 | ThresholdConfigTable、PromptTemplateEditor |
| /admin/audit-logs | 审计日志 | AuditLogTable、TraceDetail |
| /admin/dashboard | 运营看板 | BusinessMetrics、AIMetrics、ReviewMetrics |

## 4. 响应式规则

| 场景 | 桌面端 | H5 手机端 |
| --- | --- | --- |
| 表格 | 完整列 + 筛选 | 卡片列表 + 关键字段 |
| 申报表单 | 左侧步骤 + 右侧内容 | 分步单列，底部固定操作栏 |
| 材料上传 | 拖拽/选择文件 | 拍照/相册/文件选择 |
| 审核台 | 分屏详情 | 政府端 MVP 以桌面优先 |
