# 运维手册

## 1. 监控指标

| 类型 | 指标 |
| --- | --- |
| 应用 | QPS、错误率、P95 延迟 |
| RAG | 检索延迟、Top1 相似度、无结果率 |
| OCR | 任务积压、成功率、低置信度率 |
| Agent | 调用次数、失败率、转人工率 |
| 模型 | 调用量、P95 延迟、错误率、结构化输出失败率、token 成本 |
| 工具 | 工具调用次数、超时率、重试次数、错误码分布 |
| Checkpoint | checkpoint 写入失败率、恢复次数、恢复失败率 |
| 业务 | 申报量、补正率、审核超时率 |

## 2. 告警规则

| 告警 | 触发 |
| --- | --- |
| API 错误率高 | 5 分钟错误率 > 5% |
| 问答延迟高 | P95 > 5s 持续 10 分钟 |
| OCR 积压 | 队列积压超过阈值 |
| 人工兜底超时 | 普通任务超过 1 个工作日 |
| 存储异常 | 文件上传失败率升高 |
| 模型格式错误 | JSON 输出失败率连续 10 分钟超过阈值 |
| 工具超时 | 核心工具超时率持续升高 |
| Checkpoint 失败 | LangGraph checkpoint 写入失败 |

## 3. 日志查询

- 按 trace_id 查询一次用户请求。
- 按 application_id 查询申报全链路。
- 按 material_id 查询 OCR 结果。
- 按 actor_id 查询审计操作。
- 按 trace_id 查询模型调用、工具调用、checkpoint 和业务表写入链路。

## 4. 备份恢复

| 操作 | 要求 |
| --- | --- |
| 数据库恢复 | 支持按天恢复到测试环境验证 |
| 文件恢复 | 支持按 application_id 找回材料 |
| 配置回滚 | 模型阈值、Prompt、政策版本可回滚 |

## 5. 应急处理

| 场景 | 处理 |
| --- | --- |
| AI 输出异常 | 关闭对应 Agent，降级到人工兜底 |
| 模型网关异常 | 切换候选模型或降级到人工兜底 |
| 工具调用持续失败 | 熔断对应工具，保留草稿并提示稍后重试 |
| Checkpoint 恢复失败 | 以业务表为事实源恢复关键状态，记录人工处理 |
| OCR 服务不可用 | 允许材料人工录入字段 |
| 数据库异常 | 启动备份恢复流程 |
| 权限误配 | 临时冻结角色，审计排查 |
