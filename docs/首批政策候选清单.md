## 2. 候选入库流程

1. 脚本生成候选池和来源链接。
2. 政策管理员人工核验标题、原文、责任部门、有效期、申报条件和材料要求。
3. 后台人工导入已核验政策。
4. 导入后进入 RAG 知识库。
5. AI 仅生成结构化规则草稿。
6. 政策管理员审核发布后进入规则库。
7. Agent 决策时只能使用已发布知识库 + 已审核规则库。
