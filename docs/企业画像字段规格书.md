# 企业画像字段规格书

> 口径：MVP 企业手填为主，后台可导入，未来再接权威数据源。

## 1. 字段清单

| 字段名 | 中文名 | 类型 | 必填 | 来源 | 校验规则 |
| --- | --- | --- | --- | --- | --- |
| enterprise_name | 企业名称 | string | 是 | 手填/导入 | 2-100 字 |
| credit_code | 统一社会信用代码 | string | 是 | 手填/营业执照 OCR | 18 位统一社会信用代码格式 |
| industry | 所属行业 | enum | 是 | 手填/导入 | 家具制造、木材加工、商贸流通、其他 |
| enterprise_scale | 企业规模 | enum | 否 | 手填/导入 | micro、small、medium、large |
| registered_address | 注册地址 | string | 否 | 手填/OCR | 省市区地址 |
| business_scope | 经营范围 | text | 否 | 营业执照 OCR | 最长 2000 字 |
| revenue_range | 年营收区间 | enum | 否 | 手填/导入 | <500万、500-2000万、2000万-1亿、1亿以上 |
| employee_count | 员工数 | number | 否 | 手填/导入 | >= 0 |
| tax_amount | 年纳税额 | number | 否 | 手填/导入 | >= 0，单位元 |
| export_amount | 年出口额 | number | 否 | 手填/导入 | >= 0，单位元 |
| tech_upgrade_status | 技改状态 | enum | 否 | 手填 | none、planned、in_progress、completed |
| has_ip | 是否有知识产权 | boolean | 否 | 手填 | true/false |
| contact_name | 联系人 | string | 是 | 手填 | 2-30 字 |
| contact_phone | 联系电话 | string | 是 | 手填 | 手机号格式 |

## 2. 来源优先级

| 来源 | 优先级 | 说明 |
| --- | --- | --- |
| 企业手填 | P0 | MVP 主来源 |
| 后台导入 | P1 | 由管理员批量导入 |
| 营业执照 OCR | P1 | 只用于企业名称、统一社会信用代码、地址、经营范围辅助校验 |
| 权威外部数据 | P2 | 工商、税务、人社等后续接口 |

## 3. 画像快照规则

- 企业画像可持续编辑。
- 申报提交时固化为 `EnterpriseProfileSnapshot`。
- 历史申报仅引用提交时快照，不随画像后续变更而变化。
- 若画像变更影响草稿资格预判，系统需提示重新预判。

## 4. 脱敏规则

| 字段 | 脱敏方式 |
| --- | --- |
| contact_phone | 138****1234 |
| credit_code | 保留前 4 位和后 4 位 |
| tax_amount | 非授权角色展示区间 |
| revenue_range | 可展示区间，不展示明细营收 |
