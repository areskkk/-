# 字段路径映射表

> 用途：统一规则 DSL、OCR、表单预填、资格预判、审核比对中的字段路径。  
> 口径：`field_key` 必须从本表选择；新增字段需先补本表，再进入规则库。

## 1. 命名规则

- 企业画像字段使用 `enterprise_profile.*`。
- 本次申报字段使用 `application.*`。
- 申报子项字段使用 `application_item.*`。
- 材料文件字段使用 `materials.{material_type}.*`。
- OCR 识别字段使用 `ocr.{material_type}.*`。
- 系统人工确认字段使用 `manual.*`。

## 8. 字段引用约束

- DSL 中使用 `$enterprise_profile.enterprise_name` 表示引用运行时字段值。
- OCR 字段置信度低于阈值时，不得作为硬证据直接通过规则。
- `manual.*` 字段必须有操作人、时间、原因和审计记录。
- 字段废弃时不得直接删除，应标记 deprecated 并保留历史规则兼容。
